WM 2026 – Turnierbaum & Tippspiel
=================================================

WELCHE DATEI WOFÜR?

1) Turnierbaum WM 2026 (live, auto-update).html   ⭐ EMPFOHLEN
   - Eine einzige Datei, läuft per Doppelklick (offline-fähig).
   - Lädt die echten Ergebnisse automatisch aus dem Internet nach
     (beim Öffnen, alle 5 Minuten, beim Zurückwechseln zum Tab).
   - Voraussetzung fürs Auto-Update:
       * Die Datei "wm2026-results.js" muss im ÖFFENTLICHEN Repo liegen:
         fasttrack-design/claude-artifacts  (Branch: main, im Hauptordner)
       * Adresse, die abgefragt wird:
         https://raw.githubusercontent.com/fasttrack-design/claude-artifacts/main/wm2026-results.js
   - Ohne Internet / solange die Datei dort fehlt: läuft als reines
     Tippspiel mit dem eingebetteten Stand weiter.

2) Turnierbaum WM 2026 (offline).html
   - Eine einzige Datei, KEIN Auto-Update (eingefrorener Stand).
   - Ideal zum Verschicken/Archivieren oder als reines Tippspiel.

3) Turnierbaum WM 2026.html  +  wm2026-results.js   (zwei Dateien!)
   - Klassische Variante. Die HTML-Datei liest die DANEBEN liegende
     Datei "wm2026-results.js" beim Öffnen ein.
   - WICHTIG: Beide Dateien müssen im SELBEN Ordner liegen.
   - Wird "wm2026-results.js" mit echten Ergebnissen befüllt, sind sie
     beim nächsten Öffnen automatisch im Baum.


SO FUNKTIONIERT DAS TIPPSPIEL
- Sechzehntelfinale: 32 Mannschaften über die Auswahllisten eintragen.
- Klick auf eine Mannschaft = sie kommt weiter. Linien zeigen den Weg
  bis ins Finale; die nächste Runde füllt sich automatisch.
- Tore können optional eingetragen werden.
- Bestätigte (echte) Ergebnisse überschreiben Tipps und sperren das
  Feld (Schloss-Symbol). Eigene Tipps bleiben sonst gespeichert.
- "Tipps zurücksetzen" löscht nur die eigenen Tipps.

Farben: Rot & Grün
